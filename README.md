# PyRealWeapons
## Description
Python package with description a many real weapons.
## Varibales
`WeaponName`, `WeaponShotType`, `WeaponCountry`, `WeaponYear`, `WeaponBulletsType`, `WeaponOneMagazine`, `WeaponLenght`, `WeaponBarrelLenght`.

- FAMAS F1
- Franchi SPAS-12
- MAC-10
